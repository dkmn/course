# Course

Allows any content-type to be recognized as an e-learning course.

### Features

    Course object API (Drupal LMS)
        Allow any Drupal node to be part of a course requirement workflow
        Built in support for several course objects such as Quiz [1], poll
    Course API to allow access to taking courses/enrolling into courses
    External LMS support such as Moodle [2]
        Built-in Moodle/Drupal course integration, SSO
    Views [3] integration, including several default views for course listings and user status
    Integration with Certificate [4] module
    Integration with Signup [5] module for enrollments and live courses requiring attendance

### Planned

    Integration with the Rules [6] module for access to taking courses
